# oFig

Boutique de figurines

## Objectif 

Faire des cartes et un panier en dure en HTML et CSS pure


## Charte

Font : Raleway (trouvable sur Google fonts)

Couleurs :
- Rouge : `#c80000`
- Gris foncé : `#666`
- Gris clair (bordures et catégorie): `#ccc`
- du noir pour les textes et fond blanc
